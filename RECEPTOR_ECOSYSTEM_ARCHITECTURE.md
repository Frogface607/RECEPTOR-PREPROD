## 🎯 НОВАЯ АРХИТЕКТУРА: 10 МИКРОСЕРВИСОВ

### 1. 🍽️ **RECEPTOR CORE** (Бесплатный)
**Описание**: Базовое создание техкарт  
**Endpoints**: 3 максимум  
- `POST /techcards/create` - создание простых техкарт  
- `GET /techcards/{id}` - просмотр техкарты  
- `GET /techcards/list` - список техкарт  
**Цена**: FREE (до 5 техкарт/месяц)  
**Стек**: React + FastAPI + SQLite  

### 2. 🏢 **iiko PRO MODULE** (Премиум)
**Описание**: Полная интеграция с iiko RMS  
**Endpoints**: 3 максимум  
- `POST /iiko/connect` - подключение к iiko  
- `POST /iiko/export` - экспорт техкарт в iiko  
- `GET /iiko/status` - статус интеграции  
**Цена**: $99/месяц (Revenue Share с iiko)  
**Фокус**: 69к ресторанов в базе iiko  
**Стек**: FastAPI + iiko API + MongoDB
