## 🐛 ИЗВЕСТНЫЕ ПРОБЛЕМЫ:

### 1. MongoDB
- **Проблема:** `mongod` не найден в PATH
- **Причина:** MongoDB не установлен или не в PATH
- **Решение:** MongoDB Atlas (завтра)

### 2. Долгие команды в терминале
- **Проблема:** Команды с `Start-Sleep` зависают
- **Причина:** PowerShell ждет завершения
- **Решение:** Избегать `Start-Sleep`, использовать прямые команды

---

## 🎓 УРОКИ:

1. **Не использовать Start-Sleep** в автоматизированных скриптах - они блокируются
2. **MongoDB Atlas лучше локального** для разработки - не нужна установка
3. **Emergent integration** - внутренний пакет, не критичен для локальной работы
4. **npm --legacy-peer-deps** решает большинство peer dependency проблем
