# RECEPTOR — Brand Guide v2.0

## Brand Identity

**Name:** RECEPTOR
**Tagline:** Чувствует кухню
**Tagline (EN):** Feels the Kitchen
**Domain:** receptorai.pro
**Positioning:** AI-платформа управления рестораном

## Brand Personality

**Тон:** Premium Dark — как Linear, Vercel. Строго, стильно, уверенно.
**Не:** корпоративный, скучный, панибратский
**Да:** профессиональный, современный, немного дерзкий

**Voice examples:**
- DO: "20 AI-инструментов. Один клик — готовый результат."
- DO: "Управляйте рестораном из кармана."
- DO: "RECEPTOR чувствует кухню. А вы чувствуете разницу."
- DON'T: "Мы рады предложить вам инновационное решение..."
- DON'T: "Попробуйте наш уникальный продукт!"
- DON'T: "Привет! 👋 Мы тут сделали штуку..."

## Logo Generation Prompts (Freepik Spaces)

### Concept: "Рецептор" — сенсор, который чувствует

**Prompt 1 — Premium Minimal:**
```
Premium minimalist logo for "RECEPTOR", an AI restaurant management platform.
Abstract sensor/receptor icon — clean geometric lines suggesting both a neural
receptor and a chef's knife tip. Emerald green (#10b981) on pure black (#0a0a0a).
Ultra-clean, Linear/Vercel aesthetic. No text in icon. Suitable for dark UI.
Flat vector, app-icon ready.
```

**Prompt 2 — Geometric Mark:**
```
Modern geometric logomark for tech company "RECEPTOR". Abstract shape combining
a hexagonal receptor/sensor pattern with subtle culinary elements (knife edge,
flame tip). Colors: emerald green gradient (#047857 to #34d399) on dark
background (#111827). Style: premium SaaS, minimal, recognizable at 16x16px.
No text. SVG-ready.
```

**Prompt 3 — Wordmark:**
```
Premium wordmark logo "RECEPTOR" in custom sans-serif typeface. Letters are
clean, geometric, slightly extended. The "R" has a subtle receptor/antenna
detail. Color: white on dark (#111827) with emerald (#10b981) accent on the
"R". Style: high-end tech brand like Linear or Stripe. Horizontal layout.
```

**Prompt 4 — Icon + Type:**
```
Logo design for "RECEPTOR" AI platform. Left: abstract emerald green icon
resembling a stylized sensor node with 3 connected points (neural network
meets culinary precision). Right: "RECEPTOR" in clean white sans-serif.
Dark background (#0f172a). Premium tech aesthetic. Balanced spacing.
```

**Prompt 5 — App Icon Only:**
```
App icon for premium restaurant AI platform. Abstract emerald green (#10b981)
sensor symbol on near-black (#111827). Geometric, 3 converging lines forming
a receptor shape. Ultra minimal, recognizable at 32x32. iOS app icon style
with subtle depth. No text, no food imagery.
```
