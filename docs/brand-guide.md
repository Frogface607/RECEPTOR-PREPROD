# RECEPTOR — Brand Guide

## Brand Identity

**Name:** RECEPTOR
**Tagline (RU):** AI-копилот для ресторанного бизнеса
**Tagline (EN):** AI Restaurant Co-Pilot
**Domain:** receptor.ai (target)

## Logo Concept

The logo combines a chef's hat with a neural network / signal receptor motif, suggesting AI intelligence meeting culinary expertise.

### Logo Generation Prompts (Freepik / Midjourney)

**Prompt 1 (Minimal):**
```
Minimalist logo for "RECEPTOR", an AI restaurant management platform.
Chef hat icon merged with a WiFi/signal receptor symbol.
Emerald green (#10b981) on dark background (#111827).
Clean, modern, tech-forward. SVG-ready, flat design.
```

**Prompt 2 (Detailed):**
```
Professional SaaS logo "RECEPTOR". Concept: stylized chef hat where the
top folds form a neural network or signal waves pattern. Colors: emerald
green gradient (#059669 to #10b981). Style: minimal geometric, suitable
for dark backgrounds. No text in the icon — text "RECEPTOR" in clean
sans-serif below or beside. App icon ready.
```

**Prompt 3 (Icon only):**
```
App icon for restaurant AI assistant. Geometric chef hat with circuit
board lines inside. Emerald green on dark gray. Minimal, modern,
recognizable at 32x32px. Flat design, no gradients.
```

## Typography

**Primary Font:** System font stack (Apple, Segoe UI, Roboto)
**Monospace:** JetBrains Mono / Fira Code (code blocks, data)

### Heading Sizes
- H1: 1.875rem (30px), bold — page titles
- H2: 1.25rem (20px), bold — section headers
- H3: 1rem (16px), semibold — card titles
- Body: 0.875rem (14px) — default text
- Small: 0.75rem (12px) — labels, metadata

## Voice & Tone

- **Professional but friendly** — not corporate, not casual
- **Russian-first** — all UI in Russian, technical terms can stay in English (API, OLAP, HACCP)
- **Concise** — restaurant owners are busy, get to the point
- **Confident** — "Сделано" not "Попробуем сделать"
