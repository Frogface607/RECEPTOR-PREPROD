# 🎯 СИСТЕМА ТОКЕНОВ И КРЕДИТОВ RECEPTOR

## 📋 ОБЗОР

Система токенов заменяет простой подсчет техкарт на гибкую модель кредитов, где разные операции стоят разное количество токенов. Это позволяет:
- ✅ Справедливо оценивать стоимость операций
- ✅ Давать trial период для тестирования
- ✅ Монетизировать маркетплейс рецептов
- ✅ Гибко управлять лимитами по подпискам

## 🎯 TRIAL СИСТЕМА

### Автоматический Trial
```python
# При регистрации нового пользователя:
user = {
    "subscription_plan": "trial",
    "credits": 200,  # Стартовые токены
    "trial_start_date": datetime.utcnow(),
    "trial_end_date": datetime.utcnow() + timedelta(days=7),
    "trial_used": False
}
```

### Логика Trial
1. **Регистрация** → автоматически активируется trial (200 токенов, 7 дней)
2. **Использование** → токены списываются как обычно
3. **Окончание trial**:
   - Если токены не использованы → переход на Free (50 токенов/месяц)
   - Если токены использованы → показ предложения подписки
   - Можно продлить trial еще на 3 дня (только 1 раз) → +100 токенов

### Уведомления
- За 2 дня до окончания trial → email + in-app уведомление
- За 1 день до окончания → напоминание
- После окончания → предложение подписки с бонусом (первые 3 месяца -20%)

## 🛒 МАРКЕТПЛЕЙС РЕЦЕПТОВ

### Структура
```python
recipe = {
    "id": "uuid",
    "title": "Тако с уткой",
    "author_id": "user_id",
    "author_name": "Sergey Orlov",
    "price_tokens": 30,  # Стоимость в токенах
    "category": "mexican",
    "difficulty": "medium",
    "premium": False,  # True = премиум рецепт (50+ токенов)
    "purchases_count": 0,
    "rating": 4.8,
    "preview_image": "url",
    "created_at": datetime,
    "published": True
}
```

### Монетизация
1. **Публикация рецепта**:
   - Автор получает **+20 токенов** за публикацию (разово)
   - Рецепт становится доступен для покупки

2. **Покупка рецепта**:
   - Покупатель платит токенами (10-50 в зависимости от премиальности)
   - Автор получает **50%** от стоимости в токенах
   - Платформа получает **50%** (комиссия)

3. **Премиум рецепты**:
   - Автор может пометить рецепт как премиум (50+ токенов)
   - Требуется модерация
   - Дополнительные требования: фото, видео, детальное описание

### Пример
```
Рецепт "Тако с уткой" стоит 30 токенов:
- Покупатель платит: 30 токенов
- Автор получает: 15 токенов
- Платформа получает: 15 токенов
```

### Проверка лимитов перед операцией
```python
# Middleware для проверки кредитов
async def check_credits_required(user_id: str, operation: str, cost: int) -> bool:
    user = await db.users.find_one({"id": user_id})
    
    # Проверка баланса
    if user["credits"] < cost:
        raise HTTPException(
            status_code=402,  # Payment Required
            detail=f"Недостаточно токенов. Требуется: {cost}, доступно: {user['credits']}"
        )
    
    return True

# Использование в эндпоинтах
@api_router.post("/api/v1/techcards.v2/generate")
async def generate_techcard(request: TechCardRequest):
    # Проверка кредитов
    await check_credits_required(request.user_id, "techcard_generation", 50)
    
    # Генерация техкарты
    techcard = await generate_techcard_v2(...)
    
    # Списание кредитов
    await deduct_credits(request.user_id, 50, "techcard_generation", {
        "techcard_id": techcard.id,
        "dish_name": techcard.name
    })
    
    return techcard
```

### Автоматическое пополнение (ежемесячно)
```python
async def monthly_credits_reset():
    """Запускается 1 числа каждого месяца"""
    users = await db.users.find({
        "subscription_plan": {"$in": ["free", "starter", "pro", "business"]}
    }).to_list(None)
    
    for user in users:
        plan = SUBSCRIPTION_PLANS[user["subscription_plan"]]
        credits_per_month = plan["credits_per_month"]
        max_balance = plan["credits_max_balance"]
        
        # Начисляем кредиты (с учетом лимита накопления)
        new_balance = min(
            user["credits"] + credits_per_month,
            max_balance
        )
        
        await db.users.update_one(
            {"id": user["id"]},
            {
                "$set": {"credits": new_balance},
                "$push": {
                    "credit_transactions": {
                        "id": str(uuid.uuid4()),
                        "type": "earned",
                        "amount": credits_per_month,
                        "operation": "monthly_reset",
                        "description": f"Ежемесячное пополнение ({user['subscription_plan']})",
                        "created_at": datetime.utcnow()
                    }
                }
            }
        )
```

### Списание кредитов
```python
async def deduct_credits(user_id: str, amount: int, operation: str, metadata: Dict):
    """Списать кредиты за операцию"""
    user = await db.users.find_one({"id": user_id})
    
    if user["credits"] < amount:
        raise HTTPException(
            status_code=402,
            detail=f"Недостаточно токенов. Требуется: {amount}, доступно: {user['credits']}"
        )
    
    new_balance = user["credits"] - amount
    
    await db.users.update_one(
        {"id": user_id},
        {
            "$set": {"credits": new_balance, "credits_total_spent": user.get("credits_total_spent", 0) + amount},
            "$push": {
                "credit_transactions": {
                    "id": str(uuid.uuid4()),
                    "type": "spent",
                    "amount": -amount,
                    "operation": operation,
                    "description": f"Списание за {operation}",
                    "metadata": metadata,
                    "created_at": datetime.utcnow()
                }
            }
        }
    )
```
