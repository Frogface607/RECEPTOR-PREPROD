- ❌ Не учитывает контекст

### После (векторный RAG):
- ✅ Семантическое понимание запросов
- ✅ Понимает синонимы и контекст
- ✅ Умное разбиение на чанки
- ✅ Гибридный поиск (векторный + ключевые слова + BM25)
- ✅ Более релевантные результаты

## 🎯 Как использовать

### 1. Первичная индексация

```bash
# POST запрос для индексации всех документов
curl -X POST "http://localhost:8002/api/knowledge-base/index?force_reindex=true"
```

### 2. Проверка статистики

```bash
# GET запрос для статистики
curl http://localhost:8002/api/knowledge-base/stats
```

### 3. Автоматическое использование

Система автоматически использует векторный поиск при:
- Запросах в чате ассистента
- Tool calling `searchKnowledgeBase`
- Поиске в контексте для ответов

## 📈 Ожидаемые результаты

### Пример 1: Синонимы
**Запрос:** "как хранить продукты"  
**Старый поиск:** Находит только если есть точное слово "хранить"  
**Новый поиск:** Находит "хранение", "сохранение", "температурные режимы" и т.д.

### Пример 2: Контекст
**Запрос:** "нормы для ресторана"  
**Старый поиск:** Ищет слова "нормы" и "ресторан" отдельно  
**Новый поиск:** Понимает связь между словами и находит релевантные нормативы

### Пример 3: Разные формулировки
**Запрос:** "сколько стоит мясо"  
**Старый поиск:** Может не найти, если в базе написано "цена на мясо"  
**Новый поиск:** Находит по смыслу, даже при разных формулировках

## 🔧 Технические детали

### Модель эмбеддингов
- **Модель:** `text-embedding-3-small`
- **Размерность:** 1536
- **Стоимость:** ~$0.02 за 1M токенов
- **Скорость:** Быстрая (оптимизирована для production)

### Параметры чанкинга
- **Размер чанка:** 1000 символов
- **Перекрытие:** 200 символов
- **Минимальный размер:** 100 символов

### Веса гибридного поиска
- **Векторный:** 70% (семантика)
- **Ключевые слова:** 20% (точность)
- **BM25:** 10% (классическая релевантность)

## ⚠️ Важно!

**Перед использованием необходимо проиндексировать базу знаний!**

Без индексации система будет использовать fallback (простой текстовый поиск).

Индексация занимает время (зависит от размера базы знаний и скорости API OpenAI).
