ТРЕБОВАНИЯ:
1. Формат: Markdown (.md файл)
2. Категория: [укажи: haccp/ingredients_prices/techcards/hr/finance/marketing/iiko/regional/trends]
3. Актуальность: только 2025 год
4. Фокус: российский рынок
5. Структура: используй заголовки (##, ###), списки, таблицы

7. Добавь практические примеры и рекомендации
