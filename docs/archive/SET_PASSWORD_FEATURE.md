3. ✅ Модальное окно для установки пароля

## 🎯 КАК ИСПОЛЬЗОВАТЬ

### Для пользователей без пароля:

1. Открой личный кабинет
2. В секции "Подписка и аккаунт" увидишь кнопку **"🔐 Установить пароль"**
3. Нажми кнопку
4. Введи пароль (минимум 6 символов)
5. Подтверди пароль
6. Нажми "Установить пароль"
7. После установки пароля сможешь входить через email/password ✅

---

## 📊 API ENDPOINT

### POST `/api/user/{user_id}/set-password`

**Request:**
```json
{
  "password": "your_password_here"
}
```

**Response:**
```json
{
  "success": true,
  "message": "Password set successfully"
}
```

**Ошибки:**
- `404` - User not found
- `400` - User already has a password
- `400` - Password must be at least 6 characters
- `500` - Password hashing not available
