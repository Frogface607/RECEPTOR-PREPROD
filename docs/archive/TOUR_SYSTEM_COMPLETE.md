### **2. Конфиги туров `tourConfigs.js`** ✅
**Файл:** `frontend/src/tours/tourConfigs.js`

**5 туров созданы:**

#### **🏠 Welcome Tour** (главный онбординг)
- 4 шага: Приветствие → AI → IIKO → Навигация
- Показывается при первом входе
- Можно повторить через ❓ в header

#### **📝 Create Techcard Tour** (раздел Главная)
- 4 шага: Генерация → Редактирование → Маппинг → Экспорт
- **Полный флоу** от создания до экспорта в IIKO!
- Запуск: ❓ в разделе "СОЗДАТЬ ТЕХКАРТУ"

#### **🤖 AI Kitchen Tour** (раздел AI-Кухня)
- 4 шага: Профиль заведения → Генерация → Рецепт → Конвертация
- **Объясняет как профиль влияет на рецепты!**
- Запуск: ❓ в разделе "AI-КУХНЯ"

#### **🔗 IIKO Integration Tour** (опциональный)
- 4 шага: Credentials → Подключение → Синхронизация → Экспорт
- Для продвинутых пользователей
- Запуск: через кнопку ❓

#### **💰 Finance Tour** (мини-тур)
- 1 шаг: Откуда берутся цены
- Быстрое объяснение пайплайна
- Опционально

## 📝 **СОДЕРЖАНИЕ ТУРОВ**

### **📝 Create Techcard Tour - Полный флоу:**

**Шаг 1: Генерация**
```
✨ Генерация техкарты
AI создаст техкарту за вас

1. Введите название (например: "Паста карбонара")
2. Нажмите 'Создать техкарту'
3. Дождитесь генерации (15-30 секунд)

💡 Система автоматически рассчитает КБЖУ, 
   себестоимость и подберет ингредиенты
```

**Шаг 2: Редактирование**
```
✏️ Редактирование ингредиентов
Настройте под ваш рецепт

✓ Клик на количество → изменить вес
✓ Кнопка '➕' → новый ингредиент
✓ Кнопка '🗑️' → удалить
✓ Автоматический пересчет

💡 Все изменения сохраняются автоматически
```

**Шаг 3: Маппинг (если IIKO)**
```
🔗 Маппинг с IIKO
Связываем с вашей номенклатурой

1. Нажмите 'Автомаппинг'
2. Проверьте совпадения (зеленые >90%)
3. Примените изменения

💡 Без IIKO используется встроенный каталог
```

**Шаг 4: Экспорт**
```
📤 Экспорт техкарты
Выгрузите в нужном формате

✓ Excel - редактирование и печать
✓ PDF - архив и документооборот  
✓ ГОСТ-печать - официальная форма
✓ IIKO - прямая загрузка (если подключено)

💡 Экспорт в IIKO создаст товары автоматически
```

### **🤖 AI Kitchen Tour - Про профиль заведения:**

**Шаг 1: Профиль**
```
🏪 Настройте профиль заведения
AI учтет специфику вашего бизнеса

1. Тип заведения (fine dining, кафе, бар...)
2. Кухня (европейская, азиатская, fusion...)
3. Оборудование (гриль, су-вид, коптильня...)
4. Концепция (farm-to-table, веган, стритфуд...)

💡 Чем подробнее, тем точнее рецепты!
```

**Шаг 2: Генерация**
```
✨ Создайте рецепт  
AI учтет все параметры

Пример:
"Стейк для fine dining с азиатскими нотками"
→ Wagyu стейк с мисо-глазурью и васаби-пюре
  (учтен профиль: fine dining, fusion, су-вид)

✓ Адаптирован под оборудование
✓ Учтена концепция
✓ Подробное описание
✓ Можно конвертировать в V2
```

**Шаг 3: Конвертация**
```
🔄 От рецепта к техкарте
Превратите в ГОСТ-техкарту

1. Нажмите 'Конвертировать в V2'
2. Проверьте ингредиенты (AI распарсит)
3. Получите техкарту с КБЖУ и себестоимостью
```

## 💾 **PERSISTENCE**

```javascript
// LocalStorage structure:
{
  "completedTours": ["welcome", "createTechcard", "aiKitchen"],
  "skippedTours": ["iiko"],
  "hasSeenOnboarding": "true"
}
```

**Логика:**
- Если тур **completed** - не показывается автоматически
- Если тур **skipped** - можно запустить через ❓
- Welcome tour - **auto-start** для новых пользователей
