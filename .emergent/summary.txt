Receptor Pro is an AI-powered platform for the HoReCa industry, initially designed for automated tech card creation and cost management. Key features include AI-generated tech cards with detailed components, real-time cost calculation (2025 Russian market data, inflation, regional coefficients), interactive editing, and clean PDF export without prices. PRO features like Inspiration (recipe twists), Finances PRO (financial analysis with web prices), Improve Dish (recipe enhancement), and Laboratory (experimental ideas with DALL-E 3, saving capability) were implemented. The product vision significantly evolved towards an AI-Menu-Designer, aiming to generate entire, optimized menus based on detailed venue profiles, with features like intelligent ingredient optimization, single-click dish replacement, and mass tech card generation.
