<product_requirements>
The AI-Menu-Designer (МЕНЮ ЗА 15 МИНУТ) application automates menu generation for the HoReCa sector. It evolved from Receptor Pro to focus on tech card creation, cost management, and deep integration with the IIKo API. Core features include a menu creation wizard based on Venue Profiles, free-text input, and dish quantity, standardizing measurements to grams for tech cards.

The primary goal is comprehensive IIKo integration for automated tech card/price list upload, stock tracking, sales analytics via OLAP reports, and real-time pricing. Initially, this focused on programmatic creation, deletion, and retrieval of tech cards (Assembly Charts) and creating a dedicated AI Menu Designer category in IIKo. A critical issue identified was IIKo's lack of direct API support for creating DISH type products (menu items), necessitating a manual linking workflow.

Recent requirements involved developing a TechCards v2 system for stricter validation (brutto/netto/loss, allergens, HACCP), an LLM pipeline (using gpt-4o-mini with structured outputs and safe fallbacks) for enhanced generation, and flexible export capabilities (CSV for IIKo import, XLSX, PDF, and ZIP bundles for menus).
