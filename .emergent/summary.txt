Its core function is creating technical cards, managing costs, and deep integration with the IIKo API, despite IIKo's lack of direct API support for DISH products. The application features a menu creation wizard and standardizes measurements to grams. It supports comprehensive IIKo integration for automated tech card/price list upload, stock tracking, sales analytics, and real-time pricing. The application utilizes a TechCards v2 system for strict validation (brutto/netto/loss, allergens, HACCP), an LLM pipeline using  with structured outputs and safe fallbacks, and flexible export capabilities (CSV, XLSX, PDF, ZIP). It includes a code-based cost and nutrition calculator, GOST A4 printing, and a golden test suite for CI/CD. Recent enhancements focus on enabling sub-recipes, robust catalog management, inline editing, and improving LLM generation quality and data consistency, particularly by integrating USDA nutrition data.
