# 🔄 Инструкция по очистке кэша

Если изменения не видны после деплоя, попробуйте:

## 1. Очистка кэша Vercel

1. Зайдите в Vercel Dashboard
2. Откройте ваш проект
3. Перейдите в **Settings → General**
4. Найдите раздел **"Clear Build Cache"**
5. Нажмите **"Clear"**
6. Запустите новый деплой (Redeploy)

## 2. Очистка кэша браузера

### Chrome/Edge:
1. Откройте DevTools (F12)
2. Правый клик на кнопку обновления страницы
3. Выберите **"Очистить кэш и жесткая перезагрузка"**

Или:
1. `Ctrl + Shift + Delete`
2. Выберите "Кэшированные изображения и файлы"
3. Временной диапазон: "Все время"
4. Нажмите "Очистить данные"

### Firefox:
1. `Ctrl + Shift + Delete`
2. Выберите "Кэш"
3. Временной диапазон: "Все"
4. Нажмите "Очистить сейчас"

## 3. Проверка изменений

После очистки кэша проверьте:
- Кнопка "СОЗДАТЬ ТЕХКАРТУ" должна быть **фиолетовой** (не розовой)
- Кнопка "ЭКСПОРТ" должна быть **фиолетовой** (не с градиентом)
- Кнопка "ИЗМЕНИТЬ ЧЕРЕЗ AI" должна быть **фиолетовой** (не синей)
- Кнопка "СОХРАНИТЬ ТЕХКАРТУ" должна быть **фиолетовой** (не зеленой)
- **Нет эмодзи** на кнопках

## 4. Если все еще не работает

Проверьте в DevTools (F12 → Console):
```javascript
// Проверьте, какая версия загружена
console.log(document.querySelector('button[type="submit"]')?.className)
```

Должно содержать `bg-purple-600`, а не `bg-pink-600` или `bg-gradient-to-r from-pink`.
