# 📋 Промпт для ChatGPT: Структурирование информации для RECEPTOR Knowledge Base

## Инструкция для ChatGPT

Ты помогаешь структурировать информацию для базы знаний RECEPTOR - AI-ассистента для ресторанного бизнеса.

**ТВОЯ ЗАДАЧА:**
Структурируй предоставленную информацию в Markdown формат, готовый для сохранения в базу знаний.

## ТРЕБОВАНИЯ К ФОРМАТУ

### 1. Структура документа

```markdown
# [Название темы]

**Категория:** [haccp/ingredients_prices/techcards/hr/finance/marketing/iiko/regional/trends]
**Дата актуализации:** 2025-12-XX

## Краткое резюме

[2-3 предложения о чем документ]

## Основной контент

[Структурированная информация с заголовками ##, ###]

### Подразделы
[Детальная информация]

## Практические рекомендации

[Если применимо]

## Источники

- [ссылка 1]
- [ссылка 2]
```

### 2. Правила форматирования

- Используй заголовки разных уровней (##, ###, ####)
- Используй списки (-, *) для перечислений
- Используй таблицы для структурированных данных (| столбец | столбец |)
- Выделяй важные цифры и даты **жирным**
- Используй блоки кода (```) для примеров, формул, шаблонов
- Добавляй эмодзи для визуального разделения (опционально, но аккуратно)

### 3. Категории

Определи категорию автоматически:
- **haccp** - HACCP, СанПиН, нормативы, безопасность
- **ingredients_prices** - Цены на ингредиенты, поставщики
- **techcards** - Техкарты, рецепты, блюда
- **hr** - Персонал, обучение, мотивация, KPI
- **finance** - Финансы, ценообразование, налоги, ROI
- **marketing** - Маркетинг, SMM, продвижение
- **iiko** - Система iiko, автоматизация, интеграции
- **regional** - Региональные особенности, города, округа
- **trends** - Тренды, инновации, прогнозы

### 4. Что включать

✅ Включай:
- Конкретные цифры, даты, номера документов
- Практические примеры и кейсы
- Формулы и расчеты (если есть)
- Чек-листы и шаблоны
- Актуальные ссылки на источники

❌ Не включай:
- Общие фразы без конкретики
- Устаревшую информацию (только 2025 год)
- Информацию не про Россию (если не указано иное)

### 5. Примеры хорошей структуры

#### Для нормативов (HACCP):
```markdown
## Требования HACCP

### Обязательность внедрения
- ТР ТС 021/2011 требует внедрения HACCP
- СанПиН 2.3/2.4.3590-20 (действует до 2027)
- Штрафы: до 300 000 руб. для юрлиц

### 7 принципов HACCP
1. Анализ рисков
2. Критические контрольные точки
...
```

#### Для цен:
```markdown
## Цены на ингредиенты (декабрь 2025)

| Ингредиент | Единица | Москва | СПб | Регионы |
|------------|---------|--------|-----|---------|
| Говядина | кг | 450₽ | 420₽ | 380-400₽ |
...
```

#### Для техкарт:
```markdown
## Техкарта: Борщ классический

**Выход:** 300 г
**Время приготовления:** 2 часа

### Ингредиенты
| Название | Вес (г) | Примечание |
|----------|---------|------------|
| Свекла | 100 | Свежая |
...
```

## ФИНАЛЬНЫЙ ШАГ

После структурирования:
1. Сохрани результат в файл `.md`
2. Название файла: `[категория]_[краткое_название].md`
   - Например: `haccp_normativy_2025.md`
   - Или: `ingredients_prices_dekabr_2025.md`
3. Убедись, что файл в UTF-8 кодировке
4. Проверь, что все заголовки правильно оформлены
