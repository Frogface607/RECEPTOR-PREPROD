# RECEPTOR Knowledge Base

База знаний для RAG (Retrieval-Augmented Generation) поиска.

## Структура

- `receptor_master_index.md` - Главный индекс всех документов
- `receptor_haccp_sanpin.md` - HACCP и СанПиН нормативы
- `receptor_hr_management.md` - HR и управление персоналом
- `receptor_financial_roi.md` - Финансы и ROI
- `receptor_smm_marketing.md` - SMM и маркетинг
- `receptor_iiko_technical.md` - Техническая документация iiko API

## Использование

База знаний используется для:
- RAG поиска в AI-ассистенте
- Контекстных ответов на вопросы
- Генерации документов на основе знаний

## Обновление

При добавлении новых документов:
1. Добавьте файлы в эту папку
2. Обновите `receptor_master_index.md`
3. Переиндексируйте базу (если используется векторная БД)
