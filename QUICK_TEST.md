# 🚀 Быстрый запуск тестов

## Вариант 1: Автоматический (Windows)

Просто запустите:
```bash
run_tests.bat
```

Скрипт автоматически:
1. Запустит backend сервер
2. Подождет его запуска
3. Запустит тесты

## Вариант 2: Вручную

### Шаг 1: Запустите Backend

Откройте **первый терминал**:
```bash
cd backend
python -m uvicorn server:app --host 0.0.0.0 --port 8000
```

Дождитесь сообщения:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### Шаг 2: Запустите тесты

Откройте **второй терминал**:
```bash
python test_batch_menu_generation.py
```

## Что тестируется:

✅ Генерация меню (6 блюд)  
✅ Парсинг меню на позиции  
✅ Конвертация позиции в V1 рецепт  
✅ Конвертация позиции в V2 техкарту  
✅ Батчевая конвертация нескольких позиций  
✅ Проверка статуса после конвертации  

## Ожидаемое время:

- Генерация меню: ~30-60 секунд
- Конвертация V1: ~20-30 секунд
- Конвертация V2: ~30-45 секунд
- Батчевая конвертация: ~1-2 минуты

**Общее время тестирования: ~3-5 минут**
